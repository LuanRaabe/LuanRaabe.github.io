1 - Adicionar novo usuário

Para adicionar um novo cliente à lista, escreva seu nome no campo "Nome do cliente", a data de vencimento da nota no campo de data e o valor do dividendo no campo "valor do divicendo", em seguida, precione "adicionar" para mostra-lo na tabela abaixo

2 - Calcular juros

Após ter adicionado todos as informações de clientes necessárias, precione calcular juros para averiguar todas as dívidas e suas correções

3 - Agrupar por cliente ou data

Estas opções lhe darão uma melhor visualização das dívidas as organizando por data de vencimento ou alfabética

4 - Após agupar por cliente

Clique em "Total por cliente" para ver uma forma resumida dos dados por nome de cliente
